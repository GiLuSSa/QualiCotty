    QualiCotty

    QualiCotty is a lightweight qualitative content analysis tool that runs entirely in your browser. Drop .txt, .html, or .vtt documents, build a codebook, highlight passages with codes, and explore them in Analyze mode.

    Quick guide

    • Import .txt, .html, or .vtt files by drag and drop (or a .cotty / .cottybook). The document list shows about 10 at a time and scrolls if you have more.
    • In Code mode, select text and press Enter to apply the active code, or press 1–9 / 0 for the first ten codes.
    • Use the right column to choose codes, or open Codebook to edit them.
    • In Analyze mode, filter with and/or, use | for queries (code:, text:, doctype:, persons:, keywords:, indoc:), isolate segments, or merge across documents.
    • Open the project name to save or load a .cotty file.
    • wb / wf step through recent views, or Ctrl+Alt+Z and Ctrl+Alt+Y.
